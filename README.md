# SetUp Zilla

SetUp Zilla is a starter project to Next.Js projects.

## Installation

Use the package manager [yarn](https://yarnpkg.com/) to install.

Install dependencies:

```bash
yarn
```

Run in developer mode

```bash
yarn run dev
```
